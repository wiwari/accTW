摘要

一般戶外活動，無論溪降、溯溪、獨木舟、登山、攀岩，只要過程中必須接觸溪流或溪溝，在不清楚這水域水流多少時，了解此水道上游供給水份的集水區大小，能作為初步潛在水量多少的參照。但如何不數格子就能得知集水區大小呢？

集水區觀察員 (Flow Catchment Observer) 讓你快速查詢集水區面積； 使用內政部 2020 2016 20m 數值地形高程原始資料資料，利用開放工具 Grass GIS ，推估逕流累積集水面積後，使用 GDAL/OGR 轉換並切圖為圖磚形式，提供有網路環境下快速查詢集水區的方式，對於手邊沒有實體紙本地圖者能省略繁瑣的查詢步驟， 此外，亦界接中央地質調查所提供的 API 供查閱岩質。 台灣區等高線圖磚 github 專案
